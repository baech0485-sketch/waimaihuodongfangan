# 字体说明

`wqy-microhei.ttc` 是文泉驿微米黑字体，许可证为 Apache License 2.0。

项目使用它作为 ReportLab 生成中文 PDF 时的内置可嵌入字体，避免移动端微信 PDF 预览器因缺少中文字体或未嵌入 CID 字体而显示乱码。
